# FreeDesignStore

**[freedesignstore.pages.dev](https://freedesignstore.pages.dev)** — Free design tools that run in your browser. Logos, palettes, brand kits, templates, icons. No signup, no watermarks.

## What is this?

Part of [Open Frontier](https://openfrontier.pages.dev). Every tool is a single HTML file. No dependencies, no build step, no account required.

## Tools

### Brand (`/brand/`)
- Logo Maker — type name, pick style, download SVG
- Color Palette Generator — harmonious palettes with a11y checker
- Typography Pairing — font combos with live preview
- Brand Kit Builder — assemble full brand guidelines
- Business Card Designer — templates, print-ready download
- Favicon Generator — letters/emoji to multi-size favicons

### Images (`/images/`)
- Image Resizer — platform presets, custom crop
- Icon Library — 200+ SVG icons, searchable
- Gradient Maker — linear/radial/mesh, copy CSS
- Stock Photo Library — FDS/community downloads, contributor uploads, Unsplash search

### Templates (`/templates/`)
- Social Media Templates — IG, X, LinkedIn sized
- OG Image Maker — blog/product/event templates
- Presentation Builder — slide decks, export HTML
